# Financial Data Analysis using Python

## 📌 Overview
This project focuses on analyzing financial datasets to identify trends, patterns, and insights using Python. It involves statistical analysis and visualization of time-series data.

## ⚙️ Features
- Data cleaning and preprocessing
- Time-series analysis
- Moving averages
- Return calculation
- Data visualization

## 🛠️ Tools & Libraries
- Python
- NumPy
- Pandas
- Matplotlib

## 📊 Results
- Trend visualization of stock prices
- Rolling averages
- Daily returns analysis

## ▶️ How to Run
```bash
pip install -r requirements.txt
python Financial_DataAnalysis_.py
